# Jogosultság és meta függvények

| Függvény        | Jelentés                        | Példa                         |
|-----------------|---------------------------------|-------------------------------|
| `CURRENT_USER`  | Bejelentkezett user neve        | `SELECT CURRENT_USER;`        |
| `SESSION_USER`  | Session user neve               | `SELECT SESSION_USER;`        |
| `VERSION()`     | PostgreSQL verzió               | `SELECT VERSION();`           |
