# Számtani függvények

| Függvény        | Jelentés               | Példa                        |
|-----------------|------------------------|------------------------------|
| `ROUND(x, n)`   | Kerekítés `n` tizedesre| `ROUND(3.14159, 2)` → `3.14` |
| `FLOOR(x)`      | Lefele kerekítés       | `FLOOR(5.9)` → `5`           |
| `CEIL(x)`       | Felfelé kerekítés      | `CEIL(5.1)` → `6`            |
| `ABS(x)`        | Abszolút érték         | `ABS(-7)` → `7`              |
| `POWER(x, y)`   | Hatványozás            | `POWER(2, 3)` → `8`          |
| `SQRT(x)`       | Négyzetgyök            | `SQRT(9)` → `3`              |
| `RANDOM()`      | Véletlen szám (0-1)    | `RANDOM()` → `0.2342`        |
| `TRUNC(x, n)`   | Levágja a tizedeseket  | `TRUNC(3.14159, 2)` → `3.14` |
